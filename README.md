# Overview 
This simulator is programmed for "ELEMAG2026;" computes a numerical integral, shows a magnetic field as a vector graph.
# Requirement
- Windows 11
- Visual Studio 2022
# Installation
Clone this project

```bash
git clone https://github.com/yuki11011/CoilSimulator.git
```

Move to the project directory

```bash
cd CoilSimulator
```

Build the project using `msbuild`

```bash
msbuild CoilSimulator.sln /p:Configuration=Release
```
